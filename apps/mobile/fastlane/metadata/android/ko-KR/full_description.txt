AI 코딩 에이전트는 이제 기능 하나를 처음부터 끝까지 자율적으로 구현할 수 있습니다. 개발자의 역할은 승인, 답변, 리뷰처럼 중요한 판단을 내리는 일입니다. 그 판단에 꼭 노트북이 필요한 것은 아닙니다. 휴대폰이면 충분합니다.

CC Pocket을 사용하면 휴대폰에서 Codex와 Claude 세션을 시작하고, 이동 중에도 필요한 결정을 내리며, 어디서든 코드 배포까지 이어갈 수 있습니다.

이 앱은 Anthropic 또는 OpenAI와 제휴하거나, 승인 또는 보증을 받은 제품이 아닙니다.

<b>휴대폰에서 세션 시작</b>
노트북을 열지 않고도 휴대폰에서 직접 코딩 세션을 시작하세요. 프로젝트를 선택하고 권한 모드를 설정하면 몇 초 안에 작업을 시작할 수 있습니다.

<b>승인 요청을 한곳에서</b>
여러 에이전트가 동시에 실행 중이어도 괜찮습니다. 모든 세션의 대기 중인 승인 요청을 한 목록에서 확인하고, 파일 편집, 명령 실행, 도구 호출을 화면 전환 없이 처리하세요.

<b>모바일에 최적화된 UI</b>
터치 조작에 맞춰 설계되었습니다. 데스크톱 화면을 단순히 줄인 것이 아니라, 에이전트 질문에 답하고 도구 실행을 승인하며 피드백을 보내기 위한 모바일 경험입니다.

<b>풍부한 프롬프트</b>
Markdown 형식과 자동 완성되는 글머리표 목록으로 더 편하게 작성하세요. 클립보드나 갤러리에서 이미지를 첨부해 에이전트에 시각적 컨텍스트를 제공할 수 있습니다.

<b>세션 정리</b>
세션에 이름을 붙이고 프로젝트별로 관리하세요. 검색과 프로젝트 필터로 과거 대화를 빠르게 찾을 수 있습니다.

<b>diff 리뷰</b>
구문 강조가 적용된 diff로 모든 코드 변경을 한 번에 확인하세요. 파일 통계, 특정 변경 블록 선택, 피드백 전송을 지원합니다.

<b>실시간 스트리밍</b>
에이전트가 생각하고 코딩하는 과정을 실시간으로 확인하세요. 인라인 diff, 사고 과정 블록, 구조화된 계획이 스트리밍됩니다.

<b>푸시 알림</b>
에이전트가 승인을 요청하거나 작업을 완료하면 즉시 알림을 받습니다. 중요한 권한 요청을 놓치지 마세요.

<b>선택적 Supporter</b>
CC Pocket은 계속 무료로 사용할 수 있습니다. 지속적인 개발을 후원하고 싶다면 선택적 일회성 및 월간 Supporter 구매를 이용할 수 있습니다. 후원 화면에서는 구매 이력 요약을 확인할 수 있고, 월간 Supporter는 대체 앱 아이콘 혜택도 사용할 수 있습니다.

<b>유연한 연결</b>
저장된 컴퓨터, QR 코드, mDNS 자동 검색, 수동 URL 입력으로 셀프 호스팅 Bridge Server에 연결하세요. SSH를 통한 원격 서버 관리도 선택적으로 지원합니다.

<b>개인정보 우선</b>
코드와 대화는 사용자의 서버에 남습니다. 텔레메트리도, 추적도 없습니다. CC Pocket은 개인 데이터를 수집하지 않습니다.

<b>사용 방법</b>
1. Bridge Server 설치: npx @ccpocket/bridge@latest
2. 앱에서 QR 코드를 스캔하거나 URL 입력
3. 휴대폰에서 코딩 시작

5분 안에 준비할 수 있습니다.

<b>필수 조건</b>
• Codex 또는 Claude Code CLI가 설치된 Mac 또는 Linux 컴퓨터
• 휴대폰과 서버 사이의 네트워크 연결(로컬 네트워크 또는 Tailscale 권장)
